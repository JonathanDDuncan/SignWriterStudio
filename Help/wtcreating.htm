Creating a new SignWriter Studio™ File Walkthrough

To Create a new empty SignWriter Studio™ file (.SWS) which holds the Dicitionary and the SignList.

  1. Click on File/New SignWriter Studio™ File (or Ctrl-N).
  2. A Save dialog box will appear prompting you to choose the folder and the name of the new file.  Click Save to choose the filename.
  3. The filename will appear at the top of the window.
  4.  You can now use the Dictionary and SignLists menus from the Module menu.

Tips

The Document menu is always available no matter whether or not you opened a SignWriting Studio™ File.

The next time you open SignWriter Studio™ the last SignWriter Studio™ File will open.  This is so that you don't have to choose the file each time you start if you only use one file.
