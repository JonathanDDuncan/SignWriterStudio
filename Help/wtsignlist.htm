Using the SignList Walkthrough

From the Main Menu, choose Module, SignLists or Ctrl-L.

Create a new SignList

  1. From the menu choose SignList, New or Ctrl+Insert
  2. Type the name of the new sign list
  3. Click Accept.

Choose an existing SignList

  1. From the dropdown list choose an existing SignList.

Create a new SubList

  1. From the SubLists menu, choose, Add or Ctrl-A
  2. Type the name of the sub list.
  3. Click Accept.

Choose an existing SignList

  1. Click on  an existing sub list

Add Signs to a SignList

  1. Click on the Yellow plus "Add new"
  2. Find a sign in the dictionary
  3. Move to the row where the sign is you want to add.
  4. Click on the Accept button at the top.
