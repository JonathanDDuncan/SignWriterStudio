Printing the SignList Walkthrough

  1. Open the SignLists
  2. From the File menu, choose Print (Ctrl-P)
  3. A dialog box will appear. 
  4. Choose the SignList you want to print from the left selection box. 
  5. Choose the SubList you want to print from the right selection box.
  6. Set the other settings as appropiate.
  7. From the File menu, choose Print Preview.... (Ctrl-Alt-P) or Print...(Ctrl-P) as appropriate.

Tips

If you entered bilingual gloss and you have the current program setting set to bilingual gloss, then both languages will print out.
