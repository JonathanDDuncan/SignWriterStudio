ImageEditor Walkthrough

Open the ImageEditor from the Dictionary or a Document

From the File menu choose, Open.

Select the image file you want to open.

Edit the image with tools in the menu.

To save the image back to the files, choose Save As(Ctrl-Alt-A).

To save the image back to the Dictionary or Document from the File menu choose Save (Ctrl-A).

Tips

If you want to resize the image

Choose Edit/Resize from the menu

In the dialog box, enter the new width you would like the image to fit into, click on the OK button

In the dialog box, enter the new height you would like the image to fit into, click on the OK button

To erase part of an image

Choose Edit/Eraser from the menu

Click and drag the mouse on the image, a white square area will be erased as you drag the mouse. Select from the menu again to erase more.

If you erased something you didn't want to, select Edit/Undo from the menu.
