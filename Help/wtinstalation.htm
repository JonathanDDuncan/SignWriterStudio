Installation Walkthrough

  1.  Download and save the program from www.SignWriterStudio.com.
  2.  Double click on the installation file to run it.
  3.  Click on Start, the Program, click on the SignWriter Studio program folder.
  4.  Click on the SignWriter Studio program. The program will start if it has not expired otherwise you will see the About dialog box which lets you register and apply a license to continue using the program.
