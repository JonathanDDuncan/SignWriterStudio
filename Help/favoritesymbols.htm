Favorite Symbols

Use this list to create your own set of favorite symbols or sign templates. Each favorite entry has to have it's own unique name.

Tips

Add the symbols you want from the All Symbols, Hand Search, General Chooser, Hand Chooser or Arrow chooser to the sign. Select the symbols in the sign that you want to add to the favorite entry.  Click on the add button or press the [Ins] key.

To find a symbol use one of the following 5 ways.

  1. Type in the lookup text in the drop down box.
  2. While in the drop down arrow box, hit ALT-Arrow-key-down, then select the lookup text of a symbol with the arrow keys.
  3. Click on the down arrow on the side of the drop down box to select the lookup text of a symbol with the mouse.
  4. Use the arrow keys to move to the entry in the list.

Move the mouse arrow over a symbol and a short description will show up.

Keyboard

General keys
F1 Favorites Help
Insert Inserts the selected symbol into favorites.
Delete Removes the symbol in the drop down box from favorites.
Enter Move over to the chooser area with the selected symbol.
Tab Move to the next control.
Shift-Tab Move to the previous control.

When in the icon list type the name of the favorite you want to find.

Mousing

  • Click on the arrow beside the drop down box to see the list of favorite names.
  • Click on an entry to add it to the sign.
