Main Menu

You can open the various modules of the SignWriter Studio™ from here.  You can also create a new file or open an existing one.  You can also set the program options and get help from this menu.

Tips

Once a file is created or opened, all changes from the Dictionary and SignList are automatically saved to this file.  Documents are not saved in this file but rather in seperate files and must be saved manually.

The currently opened SignWriter Studio™ File is displayed at the top of the window.

Keyboard

File Menu
Ctrl-T Options... Opens the program options dialog box.
Alt-F4 Close Closes the main menu and ends the program.
Module Menu
Ctrl-O Dictionary ... Opens the Dictionary module.  Where you can access and modify dictionary entries.
Ctrl-D Document ... Opens the Document module.  You can load or import SignWriting documents for printing or editing.
