SignList Print

Print out a SignList.

Tips

If Bilingual gloss is selected in the options, both gloss languages will print out for the list if available.

Keyboard

Press the Alt key to display the shorcuts to the buttons.

Menu
Ctrl-Shift-P Page Setup  
Ctrl-Alt-P Print Preview  
Ctrl-P Print  
Alt-S Image Size The size in pixels to print the images.  Default 200
Alt-N Print only new signs. Print only signs marked as not  having appeared in previous sublists of the signlist.
Alt-I Print only important signs Print only signs marked as important.
Alt-M Print list of missing signs Print out list of signs that are pending to be added to the list.
Alt-A Print all sublist Checking this setting will print the whole sign list.
Alt-L Choose SignList to print List of signlists in the SignWriter Studio™. Choose one by moving to it then clicking or pressing the space bar.
Alt-T Choose a SubList to print List of sublists in the signlist. Choose one by moving to it then clicking or pressing the space bar.  Not activated when all sublists are selected by the Print all sublist check box.
F1 Help  

Mousing

Click on the signlist and sublist you want to print.
