Options

These options control the behaviour of the program.

Tips

Keyboard

Press the Alt key to display the shortcuts.

Menu
Alt-I User Interface Language The language of the user interface. For Future version.
Alt-B Bilingual Gloss Whether or not to support to gloss languages for the dicionary.
Alt-L Sign Language The signlanguage displayed in the dictionary.
Alt-F First Gloss Language The first gloss language displayed in the dictionary.
Alt-S Second Gloss Language The second gloss language displayed in the dictionary if applicable.
